# 声音广播篇
## 全部学生听不到声音


**问题描述**：全部学生都听不到老师的声音
**问题原因**：多个原因会造成这个结果：
1、老师没有打开广播按钮
2、有另一台设备登陆老师账号
3、老师端的麦克风有问题
4、网络导致声音通道建立失败
**处理建议**：检查老师是否打开播放按钮；是否有运营登陆老师账号；开启点到的时候，右上角声纹是否正常；切换热点是否正常；上述处理无效，请拉取全量日志，声网日志，并给出问题时间点

## 部分学生听不到
**问题描述**：有部分学生听不到老师的声音
**问题原因**：这部分学生网络波动，导致通道没能收到老师的声音通道id
**处理建议**：尝试退出应用重进可以修复
